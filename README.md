# Hours | Time tracker

A minimalist time tracker that works offline.

Built with [Vue.js](http://vuejs.org/) and [Materialize](http://materializecss.com/).

### Installation
- Install `iojs` >= 2.0.0 and `npm` >= 2.0.0
- `npm install -g gulp`
- `npm install`
- `gulp serve`

# Build as Chrome extension
- `gulp build`
